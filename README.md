# Agent Skills

Репозиторий содержит набор Skills для AI-агента, предназначенных для автоматизации инженерных процессов и поддержки рабочих задач команды.

## О проекте

Agent Skills — это модульные пакеты инструкций и конфигураций, расширяющие возможности AI-агента. Они позволяют формализовать и стандартизировать повторяемые процессы, снижать количество ручной работы и повышать предсказуемость выполнения задач.

Skills могут использоваться в различных агентных средах и платформах:

* [https://agentskills.io/home](https://agentskills.io/home)
* [https://developers.openai.com/codex/skills/](https://developers.openai.com/codex/skills/)
* [https://platform.claude.com/docs/en/agents-and-tools/agent-skills/overview](https://platform.claude.com/docs/en/agents-and-tools/agent-skills/overview)

## Интеграция с n8n

В данном репозитории Skills используются для вызова через **n8n** с применением SSH Node.

Типовой сценарий работы:

1. n8n получает входные данные из внешней системы.
2. Через SSH Node выполняется вызов соответствующего Skill на удалённом хосте.
3. Результат выполнения возвращается обратно в workflow.
4. n8n обрабатывает ответ и передаёт его в следующую ноду процесса.

Такой подход позволяет:

* изолировать выполнение логики на сервере;
* использовать существующую инфраструктуру;
* централизованно управлять автоматизацией;
* интегрировать Skills в CI/CD и другие инженерные процессы.

## Структура Skill

Каждый Skill оформляется в виде конфигурации с метаданными:

```yaml
---
name: skill-name
description: Short description of what the skill does.
---
```

Skill может содержать дополнительные инструкции, шаблоны, вспомогательные скрипты и другие ресурсы, необходимые для его выполнения.

---

## Использование

Skills активируются через workflow в n8n.
Конкретная логика вызова и параметры исполнения определяются настройкой SSH Node и конфигурацией агента на целевом сервере.
